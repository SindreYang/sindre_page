5.Markdown插入代码块

要求：要在.md文件格式格式：a.用一个单引号’’’包裹段落上的一个函数或片段的代码，例：printf() b. 代码区块使用 4 个空格或者一个制表符（Tab 键）c. 用三个单引号’’’包裹段落上的一个函数或片段的代码，并指定一种语言（也可以不指定），例如：_python_ import os

6.使用#号标记

格式：##+（空格），其中，几个#代表几级标题。

7.字体

格式：文本+（空格） 或 文本+（空格）， 其中，左右两边的”*”和”_”1~3个，1个代表斜体，2个代表粗体，3个代表粗斜体。

8.分割线

格式：***/—/___+Enter， 其中，星号、减号、底线三个以上、七个以内，内行除了空格不能有 其他东西。
